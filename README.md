## DESCRIPTION 

Constructed using interconnected breadboards, this 4-bit Arithmetic Logic Unit (ALU) functions as the essential hardware core for processing binary data. By integrating a Full Adder to handle arithmetic calculations, a 2-to-4 Decoder to interpret control signals, and a 4-to-1 Multiplexer to route the resulting data, the circuit effectively executes both complex logical and arithmetic operations.
